== Installation ==

= Minimum Requirements =

* WordPress 3.0 or greater
* PHP version 5.2.0 or greater

1. Install Heyoya Voice Reviews & Comments either via the [WordPress.org plugin directory](https://wordpress.org/plugins/heyoya-voice-comments-reviews/), or by uploading the files to your server
2. After activating Heyoya Voice Reviews & Comments, go over the Comments link in the left hand side bar of your Wordpress admin and click on the Heyoya entry.
3. You will now need to register an account with Heyoya or log in with your existing account.
4. Customize the different settings on the Heyoya Settings page.
5. That's it.  You're ready to go!
